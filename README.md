# cursor-skillz

Just some vibe coding skill files I've made for use in Cursor as commands

## The Skills
- `brain-dump-to-plan`: inspired by [Spec Kit Cursor commands](https://github.com/madebyaris/spec-kit-command-cursor/tree/main/.cursor/commands), converts an unstructured plan for a product/feature into a buildable plan in Plan Mode
- `vibe-security`: inspired by [Chris Raroque's Vibe Security Skill for Claude Code](https://github.com/raroque/vibe-security-skill/tree/main/vibe-security), provides a report of security vulnerabilities commonly missed by AI and how to fix them
